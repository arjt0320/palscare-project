# Health Buddy App

A mobile-first doctor appointment booking and medical history prototype built with:

- React
- JavaScript
- Node-ready frontend structure
- Local appointment state persistence
- Mock medical records and prescriptions

## Included screens

- Home dashboard
- Doctor search
- Doctor details and booking
- Appointment tracking
- Records and medical timeline
- Patient profile

## Removed for now

- Authentication
- Supabase integration
- TypeScript
- Backend/API layer

## Run locally

```bash
npm install
npm run dev
```
